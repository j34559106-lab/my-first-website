## CSS 변수 정의
```css
:root {
  /* Primary Colors */
  --color-primary: #6A5ACD;
  --color-primary-light: #8F83DA;
  --color-primary-dark: #4A3F90;

  /* Secondary Colors */
  --color-secondary: #00BCD4;
  --color-accent: #FFFFFF;

  /* Background Colors */
  --color-bg-primary: #FFFFFF;
  --color-bg-secondary: #F3F1FC;

  /* Text Colors */
  --color-text-primary: #333333;
  --color-text-secondary: #555555;
  --color-text-muted: #888888;

  /* Interactive Colors */
  --color-button-primary: #6A5ACD;
  --color-button-hover: #5A4DAE;
  --color-link: #00BCD4;
  --color-link-hover: #0096AA;
}
```

## 컬러 사용 가이드

### Primary Colors
- `--color-primary` (#6A5ACD): 브랜드를 대표하는 메인 컬러. 페이지 배경 그라데이션, 버튼, 강조 아이콘 등 시선을 가장 먼저 끌어야 하는 요소에 사용
- `--color-primary-light` (#8F83DA): Primary보다 부드러운 톤이 필요할 때 — 배경 위 hover 효과, 보조 배지(badge), 은은한 강조 영역
- `--color-primary-dark` (#4A3F90): 진한 대비가 필요한 텍스트나 아이콘, 다크모드에서의 강조 요소에 사용

### Secondary & Accent
- `--color-secondary` (#00BCD4): Primary와 짝을 이루는 보조 컬러. 그라데이션의 반대쪽 끝, 정보성 배지, 링크 색상의 기반
- `--color-accent` (#FFFFFF): 컬러 배경 위에서 콘텐츠 영역(카드)을 분리하고 가독성을 확보하는 데 사용. 버튼 텍스트 등 컬러 배경 위 대비 요소에도 사용

### Background Colors
- `--color-bg-primary` (#FFFFFF): 카드, 콘텐츠 박스 등 실제 텍스트가 놓이는 배경
- `--color-bg-secondary` (#F3F1FC): Primary 컬러의 아주 옅은 톤. 섹션 구분, 카드가 아닌 페이지 여백 배경에 사용해 전체 톤을 통일

### Text Colors
- `--color-text-primary` (#333333): 제목(h1~h3), 가장 중요한 본문 텍스트
- `--color-text-secondary` (#555555): 본문 단락, 설명 텍스트
- `--color-text-muted` (#888888): 캡션, 타임스탬프, 부가 설명 등 우선순위가 낮은 텍스트

### Interactive Colors
- `--color-button-primary` (#6A5ACD): 기본 CTA 버튼 배경
- `--color-button-hover` (#5A4DAE): 버튼 hover/active 상태 (Primary보다 약 15% 어둡게)
- `--color-link` (#00BCD4): 텍스트 링크 기본 색상
- `--color-link-hover` (#0096AA): 링크 hover 상태 (Secondary보다 약 20% 어둡게)

## 반응형 고려사항

### 다크모드
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1E1B2E;
    --color-bg-secondary: #15131F;
    --color-text-primary: #F2F2F2;
    --color-text-secondary: #CFCFCF;
    --color-text-muted: #9A9A9A;
    --color-primary: #8F83DA;
    --color-button-hover: #A79BEA;
  }
}
```
- 다크모드에서는 Primary/Secondary 컬러를 그대로 쓰되, 배경은 어두운 톤(`#1E1B2E`)으로 전환하고 텍스트는 밝은 회색 계열로 반전
- 채도가 높은 원색(#6A5ACD, #00BCD4)을 어두운 배경 위에 그대로 쓰면 눈부심이 있으므로, 강조 요소는 light 변형(`--color-primary-light`)을 우선 사용

### 화면 크기별 고려사항
- 모바일에서는 그라데이션 배경보다 단일 배경색(`--color-bg-secondary`) 사용을 권장 — 작은 화면에서 그라데이션이 답답해 보일 수 있음
- 버튼/링크의 색상 대비는 모든 화면 크기에서 WCAG AA 기준(대비비 4.5:1 이상)을 충족하도록 `--color-text-primary`와 `--color-bg-primary` 조합을 우선 사용
- 터치 인터페이스에서는 hover 색상(`--color-button-hover`, `--color-link-hover`)을 `:active` 상태에도 동일하게 적용해 탭 피드백을 명확히 전달
